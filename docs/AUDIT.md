# Audit and diagnostics

Fast health:

```bash
bw-monitorctl doctor
```

Deep read-only audit:

```bash
bw-monitorctl audit
```

Database details:

```bash
bw-monitorctl db-check
```

Sanitized support bundle:

```bash
bw-monitorctl diagnostics
```

Release source audit:

```bash
./tools/release-audit.sh
```
